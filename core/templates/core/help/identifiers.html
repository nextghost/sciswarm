{% trans 'Using Identifiers' %}

{% blocktrans %}Sciswarm uses various unique identifiers to link authorship and citations. If the author or cited paper isn't registered in the system yet, the identifier will be converted to an external link (if supported). Once the author or paper gets registered in the system, the identifier will be converted to a link to their user profile or paper detail page in Sciswarm, through which you can also access the original external links.{% endblocktrans %}

{% blocktrans %}Identifiers have two parts: The scheme, which determines the identifier namespace and how the value will be converted to an external link, and the value, which must be unique for the given scheme and permanently associated with the given user or paper.{% endblocktrans %}

{% blocktrans %}When entering an identifier into Sciswarm, you can either select one of the supported schemes from the drop-down menu, or you can select “Other” and enter the full identifier into the text field as “prefix:value”. Supported scheme prefixes listed below will be automatically recognized as if you had selected the scheme using the drop-down menu. But you can enter any arbitrary scheme prefix as long as the corresponding identifier namespace is well known. Scheme prefixes are case insensitive.{% endblocktrans %}

{% trans 'User Identifiers' %}

{% blocktrans %}Sciswarm supports the following user identifier schemes:{% endblocktrans %}

  • {% trans 'E-mail address (prefix: mailto)' %}
  • {% trans 'ORCID iD (prefix: orcid)' %}
  • {% trans 'XMPP address/JID (prefix: xmpp)' %}
  • {% trans 'Twitter handle (prefix: twitter)' %}
  • {% trans 'Sciswarm ID (prefix: swarm)' %}
  • {% trans 'Personal website (prefixes: http, https, ftp, ftps)' %}

{% trans 'Paper Identifiers' context 'help' %}

{% blocktrans %}Sciswarm supports the following paper identifier schemes:{% endblocktrans %}

  • {% trans 'DOI (prefix: doi)' %}
  • {% trans 'ISBN (prefix: isbn)' %}
  • {% trans 'arXiv ID (prefix: arxiv)' %}
  • {% trans 'Sciswarm ID (prefix: swarm)' %}
  • {% trans 'Web URL (prefixes: http, https, ftp, ftps)' %}

{% blocktrans %}When you select arXiv scheme from the drop-down menu, the system will accept arXiv identifiers with or without the (case sensitive) “arXiv:” prefix.{% endblocktrans %}
